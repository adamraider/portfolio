Tokenpass

Tokenly is building a suite of applications that is redefining what ownership means on the web. Tokenpass is a part of that suite and is specific to Token Controlled Access (TCA), which allows users to access content on the web based on the contents of their bitcoin wallet. I designed and implemented the user interface of this product with support from the other great developers at Tokenly.

  • Sketch
  • Blade
  • JavaScript/jQuery
  • Sass
  • Vue.js

NYCDA Home

New York Code + Design Academy was in need of a web overhaul. As lead developer I planned, designed, and developed this project. It's purpose is to allow students to request information, apply, pay-for, and enroll into any of the courses offered at NYCDA. The entire process is automated and tracked for easier analytic collection.
